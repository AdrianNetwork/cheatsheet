Open Tags

    Basic content

  • <h1> : This can be set 1-6 and is for headings
  • <p> : This is for Paragraph
  • <u> : Underline text.
  • <button> : For a Button
  • <a href="content.jpg" title='the descirption of link'> : Anchor tag for links
    • The link text goes between the tags.
  • <head> : This is the beginning of all HTML which includes links and title tags
  • <body> : This is where everything else goes.
  • <div> : For different sections, but for content semantic tags should be used
  • <script> : For JavaScript
  • <span> : For special text, like the code text here.
  • <video> : Use src attribute, and Control. No alt attribute, instead you put the description between the tags
  • <audio> : Used in the same way as video

    • Lists

  • <ul> : Unordered List
  • <ol> : Ordered list (with numbers)
  • <li> : list item. Add anohter list to make a sublist.

    • Tables

  • <table> : used to define a table
  • <thead> : used to group the header content in a table
  • <tbody> : used to group the body content in a table
  • <tfoot> : used to group the footer content in a table
  • <tr> : used to define a row in a table
  • <th> : used to define a header cell in a table
  • <td> : used to define a data cell in a table
  • colspan : specifies the number of columns a cell should span
  • rowspan : specifies the number of rows a cell should span

Closed tags

  • <!-- [content] --> : This is to add a comment.
  • <br> : Line break
  • <link> : to connect CSS (see below CSS tags)
  • <img src=".filedirectory.image" alt="image description"> :Can be local or external image

Semantics Tags

    These are used to describe the content of a webpage:

  • <header> : For the top portion of website
  • <nav> : For the navigation within header.
  • <main> : Main section
  • <article> : Another type of main
  • <section> : Part of the main
  • <footer> : Footer
  • <main> : Main section
  • <aside> : Side section or side comment
  • <caption> : Caption
  • <figure> : ANy kind of figure
  • <Figcaption> : Caption to a figure

Attributes

  • href="./directory.html" : hyperlink refrence. To connect outside content.
  • src="./content.jpg" : used for images or whatever
  • alt : Alternative text to describe an image
  • rel : Related to what? Typically a stylesheet
  • ID : used for special # css IDs or for anchor tags
    • For linking internally on a page href="#idtag"
  • controls : for video
  • target="_blank" : Opens a new window with the <a> tag.

Fonts

  • <@fontface {DETAILS}> : This is set an uncommon font.
    • Ask AI for help if you need it.
    • Google Fonts
    • Adobe Fonts
  • font-family: : Type of font
  • font-size : in px or em or wahtever
  • font-weight : Bold, normal, lighter 100-900
  • font-style : italics, oblique (slanted)
  • font-decoration : underline, overline, line-through, blink
  • text-transform : uppercase, lowercase
  • line-height : 1.4 would be 1.4x text thickness
  • word-spacing : in px or em or wahtever
  • letter-space : in px or em or wahtever
  • text-align : left, right, center, or justify

Box Model

  • display: : allows you to change the display of an element
  • block; : creates a block-level element .
  • inline; : creates an inline-level element.
  • inline-block; : Combination of the other two.
  • none; : hides the element and removes it from the document flow.
  • flex; : creates a flexible container that allows its items to be laid out in any direction and easily re-ordered.
  • grid; : creates a grid container that allows its items to be laid out in rows and columns.
  • width : the width of the content box
  • height : the height of the content box
  • padding : the space between content and border
  • border : the border around the element
  • margin : the space outside the border

Folder Navigation

  • ls : lists
  • pwd : print working directory
  • cd : change directory
  • cd .. : change directory 1 up
  • cd ../.. : change directory 2 up
  • cd ../../.. : change directory 3 up and so on
  • cd ../[diretory] : change directory up and over
  • touch [filename.txt] : makefile of any type
  • rm : remove file
  • mkdir [directory] : make new directory
  • rmdir [directory] : remove directory
  • mv [directory/file] [target directory] : move file or folder
